# Quick GitHub Setup

## Fast Setup (5 minutes)

### 1. Install Git
Download from: https://git-scm.com/download/win

### 2. Run Setup Script
Double-click: `SETUP_GITHUB.bat` in the backend folder

### 3. Create GitHub Repository
1. Go to https://github.com/new
2. Name it: `data-center-news-chatbot`
3. Choose **Private**
4. Click **Create repository**

### 4. Connect and Push
```bash
cd c:\
git remote add origin https://github.com/nicks-codes/data-center-news-chatbot.git
git branch -M main
git push -u origin main
```

### 5. Open on Work Computer
In Cursor: **File → Clone Repository** → `https://github.com/nicks-codes/data-center-news-chatbot.git`
